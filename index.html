Как использовать этот инструмент

  1. Получите ваш API-токен в настройках Hugging Face
  2. Примите условия использования модели на странице модели
  3. Вставьте ваш токен в поле ниже
  4. Нажмите "Анализировать случайный отзыв" для тестирования
Ваш токен используется только для этого сеанса и нигде не хранится.
